# 指纹B-scan数据集使用说明

## 数据集结构

```
data/fingerprint-bscan/
├── raw_data/
│   ├── volume_001/
│   │   ├── volume_001_0001.npy  # B-scan数据文件
│   │   ├── volume_001_0002.npy  # B-scan数据文件
│   │   └── ...
│   └── volume_002/              # 更多体数据（可选）
├── cropped/          # 滑窗切分后的图像（自动生成）
│   ├── volume_001_x00_z00/
│   │   ├── 0000.jpg
│   │   ├── 0001.jpg
│   │   └── ...
│   ├── volume_001_x00_z01/
│   │   ├── 0000.jpg
│   │   ├── 0001.jpg
│   │   └── ...
│   ├── ...
│   └── volume_002_y00_z00/
│   │   ├── 0000.jpg
│   │   ├── 0001.jpg
│   │   └── ...
├── filelist.txt             # 文件列表（自动生成）
└── .prepared                # 预处理完成标记（自动生成）
```

## 滑窗策略

假设每个体数据由1600张1800×500 的B-scan图像组成，即`x=1800, y=1600, z=500, win_size=256, stride=128`

- 窗口数 `win_num = round((length - win_size) / stride) + 1`

### x-z面切分

- 固定y轴，沿x和z方向滑窗
- x方向: 1800 → 13个窗口
- z方向: 500 → 2个窗口  
- 每y面: 13×2=26张
- 总计: 1600×26=**41600张/体数据**

### y-z面切分

- 固定x轴，沿y和z方向滑窗
- y方向: 1600 → 11个窗口
- z方向: 500 → 3个窗口
- 每x面: 11×2=22张
- 总计: 1800×22=**39600张/体数据**

**单个体数据总计: 81200张 256×256 图像**

## 首次运行时会自动：

1. 遍历所有体数据
2. 沿x-z和y-z方向滑窗切分
3. 保存为 256×256 图像
4. 生成filelist.txt文件列表

预处理完成后，后续训练直接加载图像，无需重复切分。

## 扩展新数据

添加新的体数据到 `data/fingerprint-bscan/raw_data` 目录，删除 `.prepared` 标记文件，
下次运行时会自动重新预处理。
